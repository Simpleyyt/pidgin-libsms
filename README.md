![Pidgin短信](icons/pidgin_sms_64_c.png "Pidgin短信") Pidgin短信
==================================================================
Pidgin短信是Pidgin/empathy的一个插件。当手机收到短信进，电脑上面有提示；可以在电脑上发手机短信。让你时刻保持在电脑屏幕上，也不会错过短信。
